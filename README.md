# 🔴 Pokédex Web App

Uma Pokédex moderna desenvolvida com HTML, CSS e JavaScript consumindo a API oficial Pokémon.

Projeto desenvolvido durante o desafio da DIO, com diversas melhorias visuais, funcionais e responsivas implementadas por mim.

---

## ✨ Preview

### 📱 Interface Principal
- Listagem dinâmica dos Pokémon
- Cards estilizados por tipo
- Paginação
- Hover animations
- Layout responsivo

### 📖 Tela de Detalhes
- Informações completas do Pokémon
- Sistema de abas
- Base Stats
- Moves
- Evolution Chain
- Navegação entre evoluções
- Visual inspirado em uma Pokédex moderna
